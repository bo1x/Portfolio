Forgotten Loop

Shooter roguelike 2D con estética pixel art, inspirado en el aclamado "Enter the Gungeon". Un proyecto donde asumo el rol de programador y game designer, creando una experiencia de bullet hell desafiante y rejugable.

Descripción del proyecto

Forgotten Loop es un roguelike de acción donde el jugador debe enfrentarse a enemigos en mazmorras generadas procedimentalmente. Con mecánicas de bullet hell y una progresión basada en la repetición, cada partida ofrece una experiencia única.

Características principales

  • Mazmorras procedimentales: Implementación usando Edgar-Unity para generar niveles únicos en base a modulos
  • Bullet hell roguelike: Patrones de disparo complejos y combate desafiante
  • Arte pixel art: Estética retro con animaciones fluidas
  • Rejugabilidad: Sistema roguelike que incentiva múltiples partidas para progresar

Mi contribución

  • Programador: Programé varias partes del proyecto, como el sistema de dungeons procedimentales y sistemas de UI y vida, además de apoyar al equipo gracias a mi mayor experiencia en el ambito de la programación
  • Game Designer: Diseño de mecánicas, balanceo y progresión
  • Generación procedimentales: Integración de Edgar-Unity para crear mazmorras dinámicas
  • Sistema de combate: Implementación de patrones de enemigos y mecánicas de bullet hell

Tecnologías utilizadas

Unity C# Edgar-Unity Git 2D Pixel Art
